# Shoplexify - Express

#### This is a simple online shop API, made using [Express.js](https://expressjs.com/).

#### This project is a Express.js API version of my previous project, [Shoplexify](https://github.com/Matthew1906/Shoplexify) (made with Python Flask)

#### This app is part of this [repository](https://github.com/Matthew1906/Shoplexify-Turborepo), which is the final form of this project: A Turborepo version built with Next.js for the Frontend and Express.js for the Backend.
